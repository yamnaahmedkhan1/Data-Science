* Histogram to observe data distribution.

## Task 4: Medical Insurance Cost Prediction
### Objective: 
 Estimate the medical insurance claim amount based on personal data. 
 
### Dataset
Medical Cost Personal Dataset

The dataset contains the following features:

Age
Sex
BMI
Number of children
Smoking status
Region
Insurance charges

### Tools and Libraries Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn

### Project Steps
Load the dataset using pandas.
Explore the dataset using .shape, .columns, and .info().
Visualize relationships between variables such as age, BMI, smoking status, and insurance charges.
Convert categorical variables into numerical values.
Split the dataset into training and testing sets.
Train a Linear Regression model.
Predict insurance charges.
Evaluate the model using MAE and RMSE.
